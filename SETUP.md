# 🚀 Quick Setup Guide

## 3-Minute Setup

### 1️⃣ Get API Key (1 min)
```
Visit: https://makersuite.google.com/app/apikey
Click: Create API key
Copy: Your new API key
```

### 2️⃣ Configure .env (1 min)
```bash
# Update .env file with your key:
GOOGLE_API_KEY=AIzaSy_PASTE_YOUR_KEY_HERE
```

### 3️⃣ Launch Extension (1 min)
```bash
# In VS Code:
Press F5
```

## ✅ Verify Everything Works
